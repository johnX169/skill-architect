# skill-architect

Architecture review skill for analyzing skills, agents, prompt systems, and engineering structures.

## Contents

- `skill-architect/SKILL.md` - the Codex skill definition
- `skill-architect.zip` - packaged copy of the skill directory

## What It Does

This skill helps review prompt-engineering artifacts by mapping structure, tracing execution flow, identifying implicit dependencies, evaluating risks, and producing grounded refactoring suggestions.

## Notes

The repository intentionally excludes local Wiki notes, personal records, and OS metadata.
